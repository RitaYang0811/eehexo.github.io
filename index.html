如何使用Hexo建立筆記

  1. 安裝 Node.js 和 npm

  2. 建立新網站，並安裝必要的套件

  3. 建立新文章

    筆記會被存放在 source/_posts 目錄下
    並使用 Markdown 格式編寫

  4. 生成靜態文件

    編譯出靜態文件，存放在 public 目錄下
    可拖曳至新的檔案中，使用Live Server開啟，但這樣比較麻煩
    所以可以使用hexo server 開啟本地伺服器

  5. 啟動本地服務器

    Ctrl + C 即可停止本地伺服器

  6. 清除暫存文件

  7. 部署到遠端

    需要先設定遠端伺服器，在 _config.yml 中設定

    然後執行 hexo deploy 即可將網站部署到 GitHub Pages
